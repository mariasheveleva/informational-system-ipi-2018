Лабораторная работа:

«Создание интерактивного электронного технического руководства Autodesk Vault»

Введение

 

Компания Autodesk шагнула чуть дальше, чем выпуск приложения для существующих САПР, упрощающих разработку инструкций и руководств. Autodesk Inventor Publisher – это абсолютно самостоятельный продукт для создания интерактивной документации на основе проектов, выполненных в привычных проектировщикам САПР. Взяв за основу 3d модель, пользователь начинает оформлять различные представления изделия, добавляя размеры, описания элементов, спецификации, выносные виды. Инструменты, доступные в Autodesk Inventor Publisher, позволяют полностью оформить документацию в любом виде – от чертежей с пояснениями до видеороликов со всплывающими комментариями. 

 

 

 

 

Цель работы

  

На основе бумажного руководства по эксплуатации и 3д модели изделия, созданной в Autodsk Inventor, необходимо создать электронную версию руководства по эксплуатации в Autodesk Inventor Publisher, которая включает в себя следующие пункты

 

1.   Создание нового проекта

2.   Сборка/Разборка изделия, расположение элементов

3.   Простановка размеров

4.   Добавление комментариев и областей детального просмотра

5.   Создание спецификации на основе спецификации из Inventor

6.   Выпуск документации

 

 

  

 

Порядок выполнения работы

 

Для создания нового проекта щёлкните по главной иконке и выберите пункт New

 

 

После этого откройте файл вашего проекта в формате Inventor (*.ipt)

 

 

 

 

 

 

 

Окно Autodesk Inventor Publisher состоит из нескольких частей

1.      Tabs & Command Panels – лента с командами, сгруппированными по вкладкам

 

2.      View Cube и Navigation Bar – основные инструменты для навигации

 

 

 

3.      Graphic Screen – рабочая область

 

 

4.      Canvas Browser – браузер со списком компонентов

 

 

 

5.      Storyboard Editor  - редактор кадров

 

 

При нажатии правой кнопкой мыши на любом элементе геометрии появляется его контекстное меню

 

 

Расположение элементов

Для создания схем сборки или наглядного представления изделий можно последовательно перемещать элементы в ручном режиме или разобрать всю сборку автоматически по выбранным осям. 

Для ручного перемещения выберите элементы и нажмите инструмент Move во вкладке Home на ленте команд

 

 

 

Траекторию движения также можно отразить, нажав Trial Visibility

 

 

Для создания сложной траектории после простых прямолинейных перемещений нажмите Add Trial Elbow.

 

Для автоматического создания иерархической структуры элементов, демонстрирующих последовательность сборки, нажмите на сборку в браузере правой клавишей мыши и  в выпавшем меню выберите AutoExplode.

 

 

 

Можно регулировать расстояние между соседними компонентами в процентах и количестве шагов при разборке

 

 

 

 Простановка размеров

 

В AIP предусмотрены инструменты для нанесения линейных, угловых и диаметральных размеров. 

Линейные – нажмите во вкладке Home

 

 

Для редактирования текста, его размера и шрифта, единиц измерения, наклона, используйте панель редактирования

 

 

 

Чтобы создать угловой или диаметральный размер, выберите Angular или Radius соответственно.

 

 

 

Создание комментариев

 

Для создания текстовой выноски нажмите Callout во вкладке Home и укажите элемент, для которого необходимо создать комментарий.

 

На ленте команд появится вкладка Annotate для редактирования.

Детальный просмотр

 

Для создания выносного вида элементов используется инструмент Detail View

 

 

Выберите область, которую хотите детализировать 

 

              

 

Спецификации

 

Для создания спецификации выберите инструмент Parts List. В открывшемся окне можно указать, какие элементы будут отображаться в списке, настроить вид таблицы, добавляя столбцы с информацией о детали, включить автоматическое создание выносок с нумераций деталей, указанных в спецификации.

Секущие плоскости

 

Чтобы показать модель в разрезе, нужно создать секущую плоскость с помощью инструменты Section Front. 

 

 

 

В зависимости от расположения секущей плоскости выберите

 

 

 

 

Чтобы указать, какая часть будет видна после разреза, щелкните левой кнопкой по нужной части.

 

 

Редактор кадров

 

В редакторе кадров можно создавать группы документов, содержащие набор снимков.

Для создания нового пакеты документов выберите New StoryBoard. При двойном щелчке левой кнопкой мыши, откроются кадры, которые содержатся в пакете.

              

 

 

Для создания нового кадра нужно нажать New Snapshot в панели Home

 

 

Переключаясь между кадрами, можно менять расположение камер, отображение геометрии изделия, выноски, размеры и много другое.

Для редактирования настроек видеоролика, нужно щёлкнуть правой клавишей мыши  в меню списка документов и выбрать пункт Show Timing. В появившихся ячейках с цифрами можно выбрать время отображения каждого кадра и длительность перехода между ними.

 

 

 

 

 

 

 

 

 

 

 

 

Выпуск документации

 

Выбор и настройка форматов документации осуществляются посредством инструментов, находящихся в  категории Publish вкладки Home.
