Лабораторная работа 1:«Передача модели изделия между САПР с помощью обменного файла STEP»


Цель работы:


В ходе лабораторной работы Вы научитесь передавать модель изделия между различными САПР, в качестве которых будете использовать Autodesk Inventor 2018 Professional и КОМПАС-3D V17. В частности, Вы научитесь:

• экспортировать в обменный файл и импортировать из обменного файла данные об изделии;

• просматривать содержимое обменных файлов с помощью специальных программ просмотра.


Cценарий работы:


При разработке модели изделия с применением различных САПР первоначально происхо­дит создание модели в одной из них. Далее эта модель передается в другую систему, например, для деталировки. При применении для этого стандарта STEP необходимо предварительно сформировать обменный файл STEP, содержащий модель изделия.

В этот момент STEP-модель изделия может быть визуализирована при помощи пакета STP Viewer, а содержание обменного файла может быть отображено при помощи пакета STEP File Browser.

Для того чтобы модель изделия была загружена в другую САПР, необходимо перевести ее из обменного файла STEP в формат требуемой системы, после чего возможен просмотр и изменение модели, правда, с потерей параметризации.

После этого модель изделия может быть возвращена в исходную САПР путем последовательного перевода измененной модели изделия в STEP и преобразования STEP-модели в формат исходной САПР.
