Руководство по обновлению содержания


Привет. Эта страница - краткий гид по обновлению содержания системы.


Структура файлов системы


pages - эта папка содержит сами страницы, имеет структуру подпапок согласно разделам самой системы

css - эта папка содержит файл стилей

images - эта папка содержит картинки, имеет структуру подпапок согласно разделам самой системы

js - эта папка содержит javascript скрипты

media - эта папка содержит медиа-содержимое (например, swf файлы)


Добавление страниц


Мы постарались сделать "движок" системы пригодным для использования студентами последующих курсов, ранее же каждая группа создавала новую систему, т.к. предыдущая не была пригодна к редактированию или обновлению.


В этой системе, для добавления новых страниц нужно сделать следующее:

  1. Зайдите в папку pages/<папка необходимого вам раздела>
  2. Создайте копию любого html файла из этой папки, переименуйте его в нужное вам имя и откройте в текстовом редакторе (например, блокнот или Notepad++)
  3. Добавьте вашу информацию между комментариями <!-- Начало содержимого страницы --> и <!-- Конец содержимого страницы --> (см. пример ниже)
    <!DOCTYPE html>
    <html>
    <head>
    <title>Информационная поддержка жизненного цикла изделия - информационная система</title>
    <link href="../../css/style.css" rel="stylesheet" type="text/css">
    <script src="../../js/jquery-2.0.3.min.js"></script>
    <script src="../../js/main.js"></script>
    </head>

    <body>
    <div id="content">
    <!-- Начало содержимого страницы -->

    <h2>Добро пожаловать!</h2>
    <br/>
    <p>Данная система рассказывает об ИПИ технологиях, а также содержит информацию по их внедрению и применению. Здесь вы узнаете о том, как изменяется процесс проектирования, при внедрении самых современных технологий.</p>
    <br/>
    <p class="image"><img src="../../images/main/0.jpg" width="50%" /></p>

    <!-- Конец содержимого страницы -->
    </div>
    </body>
    </html>
  4. Последним шагом, вам нужно добавить ссылку на новую страницу в файл index.html, который находится в той же папке. Это легко делается по аналогии со ссылками на соседние страницы.

Полезное


  1. Если вам нужно добавить на страницу изображение, то поместите его в папку images, в подходящую подпапку и обращайтесь к нему как показано в примере выше: <p class="image"><img src="../../images/main/0.jpg" /></p>
  2. Вы можете использовать "относительные пути" при указании в HTML коде путей к картинками/скриптам и т.д. Например, страница /pages/example/index.html может получить доступ к картинке /images/main/0.jpg по относительному пути ../../images/main/0.jpg. Каждый блок "../" указывает браузеру, что нужно поднятся вверх по иерархии папок. См. пример с картинкой выше.
  3. Для редактирования страницы вы можете использовать what you see is what you get редакторы, например Adobe Dreamweaver
  4. Если вам нужно изменить "шапку" или "подвал" страницы (например, верхнее меню, или текст "НГТУ им. Р.Е. Алексеева, 2014"), то вам необходимо отредактировать файл js/main.js.
  5. К сожалению, некоторые из страниц (особенно те, что были перенесены из более старых систем) нарушают общую структуру файлов системы (например, javascript файлы от раздела "Тесты" содержатся в папке /pages/tests, хотя по-правильному их нужно было поместить в папку js), но это не влияет на работу системы.
